# Freelancer Quote Studio

A portfolio-ready front-end project that solves a simple business problem: helping freelancers and small agencies create accurate quotes quickly, then save and search quote history.

## Why this project is valuable for front-end portfolios

This repository demonstrates practical front-end competencies recruiters expect for entry-level roles:

- Semantic, accessible HTML structure and ARIA live feedback
- Responsive UI with modern CSS tokens and system dark/light theme support
- Client-side state management using plain JavaScript
- Form validation and user-friendly error handling
- Dynamic rendering of tables and searchable list views
- Local persistence with `localStorage`
- Import/export workflow with JSON files
- Data handling with safe DOM updates (`textContent`, `createElement`) to reduce XSS risk

## Business problem solved

Freelancers often quote work manually in spreadsheets or messages, which is error-prone and hard to track. This app provides a quick, browser-based quote workflow with:

- Line-item quote builder (service, quantity, unit price)
- Automatic subtotal, tax, and total calculation
- Save/load/delete quote history
- Search by client or email
